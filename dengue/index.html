Project Description

The objective of this project is to predict the number of dengue cases in Singapore based on the following factors:

  • Rainfall measurements of various weather stations in Singapore
  • Temperature measurements of various weather stations in Singapore
  • Population growth/changes in Singapore
  • Time Series effects

The model was developed using Decision Trees and Neural Networks, and will forecast the weekly number of dengue cases up to eight weeks into the future.
